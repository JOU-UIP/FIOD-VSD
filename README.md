## Environment and Project Execution Steps 

### 1. **Dependencies Installation**  

```bash
pip install -r requirements.txt
```

### 2. **Project Execution**  
Clone the repository and run the main script:

```bash
git clone https://github.com/JOU-UIP/FIOD-VUE.git
cd FIOP-VUE

```
